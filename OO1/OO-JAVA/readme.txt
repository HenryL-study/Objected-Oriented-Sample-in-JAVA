1.本程序能够处理的多项式项数总个数不超过1000。系数范围-10^8<c<10^8, 指数范围0<=n<10^6。

2.输入的格式：{(a1,n1),(a2,n2),...} + {(a3,n3),(a4,n4),...} + ...
	注：
	1.所有空格以及制表符会被自动忽略。
	2.支持首项前面有符号。
	3.不支持此类型空串：  {(,)}  {}   {()}
	4.输入中系数可为0，系数为0的项不会输出，如{(0,1)}，所以如果只输入{(0,0)}不会输出任何项。
	5.支持同一多项式中有同类项，将会自动合并。
	6.不支持小数。
	7.不支持连续  "," "+"。
	8.不支持的格式会提示输入错误。

3.导入说明：Eclipse -> File -> Import -> 选择路径 ->导入 ->直接运行（控制台非命令行）。
