工程文件可以直接从eclipse导入。

1.输入格式为控制台输入，每行一条请求，以run回车结束。数字允许有前0和正负号，不允许小数，但是请求若负的楼层和时间则会
被忽略。时间太太则该指令会被忽略。如：
(FR,3,DOWN,0)
(FR,1,UP,1)
(ER,1,2)
(ER,6,4)
run
注：允许有空格，默认最大请求数量为1000。超过1000的请求程序将不进行处理。注意过于大量的输入可能会导致控制台管道关闭。

2.如果输入不能满足时间排序（大于等于），程序将会结束不进行调度。如果是错误的调度请求（时间为负数包含在此,程序将会先判断是否是
错误的调度请求），则会自动忽略该条请求。

3.输出时间和方向说明。输出时间t为电梯初始状态到电梯停止状态用时（不包括停止时的开关门时间）。
方向为电梯到这层楼的方向或者外部输入着想去的方向（对应两种不同的请求），如果两次楼层不变。方向不变。
样例输出（以第一条的输入为例）
(3, DOWN, 1.0)      1->3  2*0.5
(1, UP, 3.0)	    3->1  1+1(前一次开门时间)+t(3->1) = 3.0s
(1, UP, 4.0)
(6, UP, 7.5)
